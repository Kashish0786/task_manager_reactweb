# Task Manager React Web

A **frontend-only Task Manager application** built with **React JS** and **Tailwind CSS**.  
This project demonstrates strong frontend development skills including 
**UI design**, **data persistence**, and **responsive design**. 
---

## 🚀 Features
- Create task, New task, and Complete tasks
- Persistent data using `localStorage`
- Separate views for Employee and Admin dashboards
- Logout functionality
- Fully responsive design for mobile and desktop
- Modern UI with Tailwind CSS

---

## 🛠️ Technologies Used
- **React JS** – Functional components, `useState`, `Context API`
- **Tailwind CSS** – Modern responsive styling
- **JavaScript** – DOM manipulation, state management
- **localStorage** – Frontend data persistence

---

## 📈 Learning Outcomes / Skills Demonstrated
- Strong **React JS fundamentals** (functional components, hooks)
- Using **Context API** for state and authentication management
- Implementing **persistent storage** with `localStorage`
- Building **responsive UIs** with Tailwind CSS
- Creating **dynamic dashboards** for multiple user roles
